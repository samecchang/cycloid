# The Elegance of Cycloid Curves

> **A dedicated single-page web tool showcasing the mathematical beauty of cycloid trajectories.**

## The Backstory
This project is a modern tribute to a childhood fascination. Years ago, I wrote a program on an **Apple II** to calculate and draw cycloid curves. I had hoped to enter it into a science and technology fair, but for various reasons, the project was never submitted. 

Decades later, I’ve revisited this concept. Moving beyond the constraints of 8-bit computing, this tool uses modern mathematical algorithms to render these mesmerizing patterns with fluid precision.

## What is a Cycloid?
If you ever played with a **Spirograph** as a child—placing a pen into a small geared wheel and rotating it inside a larger one—the path left behind by your pen was a **cycloid**. 

Mathematically, a cycloid is the curve traced by a point on a circle as it rolls along another curve or a straight line. This tool allows you to simulate that physical motion digitally, recapturing the magic of a pen on a gear.

## Features
*   **Dynamic Parameter Tuning:** Adjust the diameter of the rolling circle in real-time to see how it alters the geometry.
*   **Custom Point Selection:** Choose any point relative to the rolling circle to generate unique variations of the curve.
*   **Fluid Animations:** High-refresh-rate rendering that captures the "liquid" motion of the gears.
*   **Single-Page Utility:** A minimalist, lightweight interface designed for focus and exploration.

## How to Use
1.  **Adjust the Slider:** Change the **Small Circle Diameter** to redefine the scale of the curves.
2.  **Pick a Point:** Select the "pen position" on the rolling circle.
3.  **Visualize:** Watch as the tool calculates and draws the intricate mathematical path instantly.

## Mathematical Logic
While my original Apple II version relied on basic iterative steps, this modern version utilizes refined parametric equations to ensure perfect accuracy and smooth curves at any resolution.
